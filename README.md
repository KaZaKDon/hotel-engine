# Hotel Engine

Интерактивная система бронирования гостиницы с визуальным выбором номера.

---

## 🚀 Возможности

- выбор этажа
- план здания
- интерактивный план комнаты
- выбор кровати (визуально)
- фото и видео номеров
- заявка на бронирование
- отправка в Telegram / email
- демо-админка
- отчёты (демо)

---

## 🧱 Стек

- React + Vite
- PHP (API)
- PHPMailer
- CSS Variables

---

## 📁 Структура

txt
src/
  data/
  screens/

  styles/

  Основные экраны

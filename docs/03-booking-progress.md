# Hotel Engine — текущий прогресс и план

## 📌 Общая идея проекта

Интерактивная система бронирования гостиниц с визуальным выбором номера и (в будущем) койко-места.

---

## ✅ Что уже реализовано

### 🏗 Архитектура

* React + Vite
* Модульная структура
* Data-driven подход (всё из конфигов)

---

### 🏨 Данные

* `hotel.config.js`

  * гостиницы
  * этажи
  * номера
  * питание
* `furniture.catalog.js`

  * каталог мебели (размеры, типы)

---

### 🗺 Навигация

```txt
Главная → Этажи → План этажа → Комната
```

---

### 🏢 Экран этажей

* Вертикальная схема здания
* 1 этаж — сервис (ресторан)
* 2-3 этаж — номера

---

### 🏨 План этажа

* Горизонтальный коридор
* Номера:

  * нечётные сверху
  * чётные снизу

---

### 🛏 Комната (RoomScreen)

#### План комнаты

* SVG `viewBox: 600x450`
* стены, дверь, окно
* мебель из каталога
* координатная система

#### Мебель

* хранится в `room.furniture`
* размеры берутся из каталога
* поддержка rotation

---

### 🛏 Кровати (интерактив)

* hover подсветка
* click выбор
* активная кровать выделяется
* tooltip (через `<title>`)

📌 Важно:

```txt
выбор кровати — визуальный
бронь — пока только номер
```

---

### 🖼 Медиа

* карусель фото
* видео (модалка)

---

### 💰 Блок бронирования

* даты
* питание
* цена
* кнопка “Забронировать номер”

---

### 🧾 Модалка заявки

* имя

* подтверждение

### 💾 Сохранение

* требует API ключ
* нужна верификация домена
* лишняя сложность для MVP

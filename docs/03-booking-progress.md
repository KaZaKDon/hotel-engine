# Hotel Engine — текущий прогресс и план

## 📌 Общая идея проекта

Интерактивная система бронирования гостиниц с визуальным выбором номера и (в будущем) койко-места.

Проект ориентирован на демонстрацию клиентам и последующую адаптацию под конкретные объекты размещения.

---

## ✅ Что уже реализовано

### 🏗 Архитектура

- React + Vite
- Модульная структура
- Data-driven подход (всё через конфиги)

---

### 🏨 Данные

- `hotel.config.js`
  - гостиницы
  - этажи
  - номера
  - питание

- разбиение на модули:
  - floors
  - rooms (по этажам)
  - beds
  - tariffs

- `furniture.catalog.js`
  - типы мебели
  - размеры
  - визуальные классы
